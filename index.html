ASL-LEX is a database of lexical and phonological properties of nearly 1,000 signs in American Sign Language. This project is a collaboration between the Laboratory for Language and Cognitive Neuroscience at San Diego State University and the Psycholinguistics and Linguistics Lab at Tufts University. ASL-LEX is a searchable database of subjective frequency ratings, iconicity ratings, lexical properties (e.g., initialized signs; lexical class), and six phonological features from which neighborhood densities have been calculated. ASL-LEX also provides English translations and, for a subset of signs, alternative translations. Lastly, the database also contains time codes for the sign onset and offset within each video clip.

Basic keyword search

Type the English translation in the search box and press enter. Information about that sign will be available in the “Sign Data” tab, and the sign node in the visual lexicon will move to the center of the screen. If more than one sign or English translation matches what you have typed, you will see a list of words to select from.

Advanced search – using filters

You can narrow down the lexicon by applying filters (e.g., requesting a range of frequency or iconicity values) on the “Filter” tab. See the instructions on how to use filters.
