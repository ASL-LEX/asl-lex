ASL-LEX


  • This is a network graph visualization for 2768 signs in the ASL-LEX 2.0 database.
  • Signs that share all but one of SelectedFingers, Flexion properties, Spread Properties, SpreadChange, ThumbPosition, ThumbContact, SignType, Movement, MajorLocation, MinorLocation, Contact, NonDominantHandshape and UlnarRotation are connected by lines (edges).
  • Please allow few seconds for the graph to load the first time.
Exploring the Lexicon


  • Click on a node to zoom into that section of the graph
  • Select a subset of nodes by dragging your mouse cursor over them when the + sign appears. Then choose "See Pair Plots". This will render an alternate visualization where you can see pairwise scatter plots of the selected signs.
  • To reset the graph to it's original state, click the Reset graph button below.
